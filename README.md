‚ÄÆVRCX is an assistant/companion application for VRChat that provides information about and helps you accomplish various things related to VRChat in a more convenient fashion than relying on the plain VRChat client (desktop or VR), or website alone. It also includes some other neat features outlined below.

# ‚ÄÆGetting Started

<div align="center">

‚ÄÆDownload and install the latest installer (`VRCX_Setup.exe`) from [here](https://github.com/vrcx-team/VRCX/releases/latest).

‚ÄÆFor macOS and Linux check [here](https://github.com/vrcx-team/VRCX/wiki/Running-VRCX-on-Linux) for more info.

‚ÄÆBeta/nightly build available [here](https://vrcx.app/github/nightly) or in-app `Settings -> General -> Change build`.

# ‚ÄÆFeatures

<div align="left">

- ‚ÄÆ :family: Friend, world, and avatar list management
    - ‚ÄÆManage your friends list, world/group/avatar lists outside of VRChat.
    - ‚ÄÆMonitor the activity of your friends and track their online status, locations, and avatars.
    - Track friendship history including add dates, time spent together, and name changes.
    - Save notes and memos to help remember how you met.
- ‚ÄÆ :bar_chart: Customizable Dashboard with widgets
    - ‚ÄÆBuild personalized multi-panel layouts with Feed, GameLog, and Instance widgets.
    - ‚ÄÆCreate multiple dashboards, each with configurable event filters and column visibility.
- ‚ÄÆ :mag: Powerful search across all entities
    - ‚ÄÆSearch for users, worlds, avatars, and groups, or paste IDs and URLs for direct access.
    - ‚ÄÆQuick Search provides instant client-side fuzzy search across your friends, avatars, worlds, and groups.
- ‚ÄÆ :chart_with_upwards_trend: Activity Heatmap
    - ‚ÄÆVisualize a user's online activity patterns with a day-of-week √ó hour-of-day heatmap, including peak stats.
- ‚ÄÆ :camera: Store world data in the pictures you take in-game, so you can remember that one world you took those cool pictures in like... 6 months ago!
- ‚ÄÆ :bell: Monitor/respond to notifications
    - ‚ÄÆYou can send/receive invites and friend requests from VRCX as well as see the instance info of invites that you receive.
- ‚ÄÆ :scroll: See stats/players for your current instance
- ‚ÄÆ :tv: See the links to videos that are playing in the world you're in, as well as various other logged data.
- ‚ÄÆ :performing_arts: Social Status Presets
    - ‚ÄÆSave and quickly apply status + status description combinations from the sidebar or user dialog.
- ‚ÄÆ:rotating_light: VRChat Server Status
    - ‚ÄÆA status bar indicator and login page alert inform you of VRChat server issues and outages in real time.
- ‚ÄÆ :bar_chart: Improved Discord Rich Presence
    - ‚ÄÆDisplay detailed instance information in Discord, including world thumbnail, name, player count, and a join button for public lobbies.
- ‚ÄÆ:crystal_ball: VR Overlay with configurable live feed of all supported events/notifications
- ‚ÄÆ :outbox_tray: Upload and manage avatar/world images and details without Unity
- ‚ÄÆ :electric_plug: Automatically launch apps when you start VRChat
- ‚ÄÆ :skull: Automatically restart and join last instance when VRC crashes
- ‚ÄÆ :left_right_arrow: Export/import data
    - ‚ÄÆ Export friends list, avatar list, Discord names, notes, and favorite groups. Import favorite groups and group moderation bans.
